利用思路

上来撸一遍程序流程，比较复杂的地方就是他自己写的pad和unpad函数了，其他都是常规rsa操作。可能是运气好，一上来就注意到了unpad函数里这句话

return ''.join(ret[0:-ord(ret[-1])])

注意这里字符串截取是用的[0:-ord(ret[-1])]，脑子马上涌出一个骚思路，于是构造payload试一下

payload因为是不可见字符，所以下面为hex编码后的

首先，选择菜单1，发送payload1，发送218个字符，最后一个字符为0xff，于是经过程序的这段逻辑后

flag就变成了flag + '\xff' * 218，然后再次选择菜单1，再次发送payload2后，flag不出意外就会变成flag[0] + 'a' * 255，那么现在未知字符只有一个了，直接爆破一下就ok了，然后就是逐字节往后爆破了。
